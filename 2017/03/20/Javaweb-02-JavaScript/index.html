Javaweb_02.JavaScript

发表于 2017-03-20   |   分类于 JavaWeb   |     |   阅读次数

概述

* 一种用于开发交互式web页面的编程语言
* 解释型语言,不需要进行编译,由浏览器执行

* 组成部分:
    1. 基本语法部分 ---> ECMAScript ---> 规范了JS使用基本语法
    2. 浏览器对象 -----> BOM ----------> Browser Object Model
        (浏览器内置对象)
    3. 文档对象 -------> DOM ----------> Document Object Model
        (HTML文档以及文档的每一个标签,都被JS当作对象)
* 学习路线:
    1. 基本语法
        (1)变量
            运算符
            流程控制
        (2)方法
    2. 面向对象
        (3)获取对象(和Java不同,Java对象大多数要自己创建,JS是获取内置好的对象)
        (4)操作对象

基本语法

变量

1.声明格式: var 变量名 = 值;
    var在JS中是一个关键字,代表变量    
2.变量名规则与规范与Java一样
3.变量值可以是任意类型,并且可以被重复赋其他类型的值
拓展:
    (1)NaN ---> not a number :JS中非数字运算得出的结果
    (2)typeof(变量) ---> 返回当前变量的数据类型

数据类型

分类: 
    1. 数字类型 ---> 整型,浮点
    2. 布尔类型 ---> true,false
    3. 字符串类型 -> "" 或 '' 包裹
    4. null
    5. undefined -> null的衍生,变量只声明不赋值时,就是undefined
        undefinerd == null :结果是true

运算符

JS特殊运算符:
    == : 比较值而不比较数据类型
    ===: 既比较值也比较数据类型

流程控制

在判断语句中,JS会将相关数据类型转成对应的boolean值,和Java不同

函数

BOM对象

概述:     
    * JS是基于对象的,对象不是自己new的,而是获取第三方内置的对象,来使用
    * 浏览器内置的对象就是BOM对象
常用对象
    1. window对象 --- 当前浏览器内置的对象
        * window.alert(); //弹出警告框
        * window.confirm(); //确认框
            var flag = confirm("");
        * window.prompt(); //输入框
            var txt = prompt("");
        凡是window对象的方法,都可以不加window前缀

DOM对象

概述:    
    * JS主要操作的是HTML中的内容
        * 操作流程:
            1. 获取要操作的标签对应的DOM对象 --- 获取
            2. 操作这个DOM对象 --------------- 调用
    * 获取div标签对象,然后获取内容体并打印:
        1. 获取对象:
            (1)先给标签添加id
            (2)获取语法: var d1 = document.getElementById("id");
            (3)注意加载顺序:
                如果script在head中,直接调用获取语法,不能获取,因为div还没加载
            解决方案:
            1. 将script实现存储在div后面
            2. 在body标签中添加属性onload = "函数"
               script中有对应的函数实现,获取div对象
            执行效果: 页面加载完(注意是加载完),调用这个函数
    * 方案一:
        <div id = "d1"> hello </div>
        <script>
        var d1 = document.getElementById("d1");
        alert(d1);            
        </script>
    * 方案二:
        <head>
            <script>
                function get(){
                    var d1 = document.getElementById("d1");
                    alert(d1);        
                }
            </script>
        </head>
        <body onload = "get()">
            div id = "d1"> hello </div>
        </body>
    * 方案三(优化):
        1. 编写函数
        2. window.onload = 方法名; (方法没有括号)
        <head>
            <script>
                function get(){
                    var d1 = document.getElementById("d1");
                    alert(d1);        
                }
                window.onload = get;
            </script>
        </head>
        <body>
            div id = "d1"> hello </div>
        </body>    
    * 方案四(优化): 匿名函数
        <head>
            <script>
                window.onload = function() {
                    var d1 = document.getElementById("d1");
                    alert(d1);        
                }
            </script>
        </head>
        <body>
            div id = "d1"> hello </div>
        </body>
