JavaWeb_01.HTML

发表于 2017-03-18   |   分类于 JavaWeb   |     |   阅读次数

概述

1.超文本标记语言
    超文本- 不仅可以存文本, 还可以为文本添加样式,还可以存视频,音频,图片
    标记- HTML源码中使用标签实现超文本存储,标签称之为标记
    语言- 一门计算机语言

2.作用: 编写静态网页
3.特点:
    1. 是一种解释型语言,源码不需要编译
        编译型: 源码-->机器码(0101...)-->以后每次执行机器码
        解释型: 源码-->每次直接执行源码
       拓展: 编译型效率更高
    2. 文件的扩展名是*.html或*.htm
    3. html结构都是由标签组成
        标签名都是预先定义好的,具有固定好的功能;
        标签名不区分大小写
        标签由开始标签和结束标签组成
        标签若没有"内容体",可以自结束(<img />也称"空标签")
4.注释: <!-- 注释内容,不能嵌套使用 -->
